# 脉微名片开放接口

### 每日海报API接口

公司的一个项目，需要每天为用户推送一张海报，而且还要根据节气、实时热点每日变化，需要精心挑选，但是搜了一遍，并没有发现这样的api，很是头疼，但是这个需求还有实现，没办法，只能找人去专门维护，目前已经维护了近100天了，之后还会一直运营下去，所以想把此接口开放，希望能帮到有需要的朋友！
